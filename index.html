1. Electrical Circuit Basics

Quantities & Units

  • Voltage (V): Electrical potential, unit = volt (V)
  • Current (I): Flow of charge, unit = ampere (A)
  • Resistance (R): Opposition to current, unit = ohm (Ω)
  • Power (P): Energy per second, unit = watt (W)

Ohm’s Law: \( V = I \cdot R \)
Power Formula: \( P = V \cdot I = I^2 R = \frac{V^2}{R} \)

Worked Example (Beginner)

Question: A bulb has resistance \( 240\,\Omega \) and is connected to a \( 12\,V \) battery. Find the current and power.

Solution:
Using Ohm's law: \( I = \frac{V}{R} = \frac{12}{240} = 0.05\,\text{A} = 50\,\text{mA}. \)
Power: \( P = V \cdot I = 12 \times 0.05 = 0.6\,\text{W}. \)

2. AC and DC Circuits

DC (Direct Current)

Constant voltage/current (e.g., batteries). Flows in one direction.

AC (Alternating Current)

Voltage changes sinusoidally: \( v(t) = V_{\text{peak}} \sin(2\pi f t) \)

RMS Voltage: \( V_{\text{rms}} = \frac{V_{\text{peak}}}{\sqrt{2}} \)

Note: For TVET CDACC exam questions, make sure to state units and show each algebraic step clearly.

3. Phasor Diagrams & Impedance

A phasor is a rotating vector representing AC voltage or current.

Impedance (\(Z\)) generalizes resistance to AC circuits:

\( Z = R + jX \), where \( X = X_L - X_C \)

\( X_L = \omega L \), \( X_C = \frac{1}{\omega C} \), \( \omega = 2\pi f \)

Magnitude: \( |Z| = \sqrt{R^2 + (X_L - X_C)^2} \)

Phasor: voltage leading current in an inductive circuit.

4. Semiconductors

Intrinsic vs. Extrinsic

Intrinsic: Pure Si or Ge. Equal electrons and holes.

Extrinsic: Doped to create charge carriers.

  • n-type: Doped with P/As → extra electrons
  • p-type: Doped with B/Ga → extra holes

Electron Breakdown

At high reverse voltage, electrons gain energy → avalanche or Zener breakdown.

5. PN Junctions & Diodes

Formed by joining p-type and n-type materials.

Forward Bias: \( V > 0.7\,\text{V} \) (Si) → current flows.

Reverse Bias: Small leakage current until breakdown.

Diode equation: \( I = I_s \left( e^{\frac{V}{nV_T}} - 1 \right) \)

✅ Note: PNP and NPN are transistors, not diodes. A diode is simply a PN junction.

6. Number Systems & Boolean Algebra (Full, step-by-step)

Binary & Decimal (Step-by-step)

Concept: Binary is base-2 (digits 0 or 1). Decimal is base-10 (0–9).

Example A — Binary to Decimal

Convert \(1101_2\) to decimal.

\(1101_2 = 1\cdot2^3 + 1\cdot2^2 + 0\cdot2^1 + 1\cdot2^0 = 8 + 4 + 0 + 1 = 13_{10}.\)
Example B — Decimal to Binary (Divide by 2 method)

Convert \(45_{10}\) to binary.

Steps (divide by 2, record remainder):
\(45 \div 2 = 22\) remainder \(1\) → LSB
\(22 \div 2 = 11\) remainder \(0\)
\(11 \div 2 = 5\) remainder \(1\)
\(5 \div 2 = 2\) remainder \(1\)
\(2 \div 2 = 1\) remainder \(0\)
\(1 \div 2 = 0\) remainder \(1\) → MSB
Read from MSB to LSB: \(101101_2\). So \(45_{10} = 101101_2.\)

Hexadecimal & Octal (Quick conversions)

Hex is base-16 (0–9, A–F). Octal is base-8 (0–7).

Example — Decimal to Hex

Convert \(254_{10}\) to hex.

\(254 \div 16 = 15\) remainder \(14\) → 14 = E in hex. \(15\) is F.
So \(254_{10} = FE_{16}.\)

Two's Complement (Signed numbers)

Two's complement is used to represent negative integers in binary.

Steps to get two's complement (\( -N \)) from positive binary:
  1. Write the positive value in binary with the required number of bits.
  2. Invert all bits (one's complement).
  3. Add 1 to the inverted result.
Example — Represent \( -5 \) in 8-bit two's complement
\(5_{10} = 00000101_2\). Invert bits: \(11111010\). Add 1: \(11111010 + 1 = 11111011_2\). So \(-5\) is \(11111011_2\) (8-bit two's complement).

Gray Code (with conversion)

Gray code formula from binary: \(G = B \oplus (B >> 1)\)

Example — Convert \(1011_2\) to Gray
Shift right: \(B >> 1 = 0101\). XOR: \(1011 \oplus 0101 = 1110\). So Gray = \(1110.\)

Boolean Algebra & Logic Gates — Quick Reference

AND: \(A\cdot B\). OR: \(A + B\). NOT: \(\overline{A}\). XOR: \(A \oplus B\).

Interactive: Decimal → Binary / Hex / Octal

Binary: 101101

Hex: 2D

Octal: 55

Interactive: Binary → Decimal

Decimal: 45

Worked Practice Questions (TVET CDACC style)

  1. Question 1: Convert \(37_{10}\) to binary and hexadecimal.
    Show solution
    \(37 \div 2 = 18\) r 1; \(18 \div 2 = 9\) r 0; \(9 \div 2 = 4\) r 1; \(4 \div 2 = 2\) r 0; \(2 \div 2 = 1\) r 0; \(1\div 2 = 0\) r 1 → binary = \(100101_2\).
    Hex: \(37 \div 16 = 2\) r 5 → \(25_{16}.\)
  2. Question 2: Represent \(-12\) in 8-bit two's complement.
    Show solution
    \(12_{10} = 00001100_2.\) Invert bits: \(11110011\). Add 1 → \(11110100_2\). So \(-12\) = \(11110100_2\) (8-bit).
  3. Question 3: Simplify using Boolean algebra: \(A + A B\).
    Show solution
    \(A + A B = A(1 + B) = A(1) = A.\)

7. TVET CDACC / ICT Basic Electronics — Past Papers & Practice

This section helps you practice with exam-style questions and step-by-step marking-style answers. Use the Print button to create a paper for timed practice.

Sample Past-Paper (Short)
  1. Define voltage, current, and resistance with units. (3 marks)
  2. State Ohm's law and calculate current for a 24 V source across a 480 Ω resistor. (3 marks)
  3. Convert the following to binary: 156. (4 marks)
  4. Represent -20 in 8-bit two's complement. (3 marks)
  5. Simplify using Boolean algebra: \( (A + \overline{B}) (A + B) \). (4 marks)
Mark Scheme / Worked Answers
Answers
  1. Voltage (V), Current (A), Resistance (Ω) — short definitions. (1 each)
  2. Ohm's law: \(I = V / R\). \(I = 24 / 480 = 0.05\,A = 50\,mA.\)
  3. 156 to binary: \(156\div2\) -> remainders -> \(10011100_2\).
  4. -20 8-bit two's complement: 20 -> \(00010100\), invert -> \(11101011\), add 1 -> \(11101100\).
  5. Simplify: \((A + \overline{B})(A + B) = A + (\overline{B} B) = A + 0 = A.\)
